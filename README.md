# GraphicClock

A graphic clock made with Java 2d.

## Requirements

Java 8 JRE or JDK

## QuickStart

- Download the [latest release](https://github.com/BertrandAlexandre/GraphicClock/releases).
- Launch `GraphicClock.jar`.
